    test_super v0.1.0 TestSuper

    Top-level module used in “Robust compute for RDF queries” post.

    The post aims to demo how to manage fault tolerance in Elixir with supervision trees. It first reviews the process model for Erlang (Elixir) and then discusses the abstraction patterns that OTP provides for processes: agents, servers, supervisors, and tasks.

    The post demos the GenServer, Supervisor, and DynamicSupervisor patterns.
