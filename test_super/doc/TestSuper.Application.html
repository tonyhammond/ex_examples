    Link to this function start(type, args)

    Application start/2 function calls_start/3 with boolean flag.

    The boolean flag arg on the _start/3 call selects a dynamic supervision tree on true, and a static supervision tree on false. Initial setting is false, i.e. selects for a static supervision tree.
