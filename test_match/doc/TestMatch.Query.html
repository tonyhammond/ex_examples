    test_match v0.1.0 TestMatch.Query

    Module providing a struct for queries.

    The %TestMatch.Query{} struct is used to keep some related data fields together. One of the fields :type tracks whether this is a property graph query (:lpg) or a semantic graph query (:rdf). The other fields are straightforward data access fields:

    • :data – query_data
    • :file – name of query_file (within the queries_dir)
    • :path – absolute path of query_file
    • :uri – absolute URI of query_file

    Examples 

    iex> data |> TestMatch.Query.new("temp.rq", :rdf)
%TestMatch.Query{
  data: "<QUERY_DATA>"
  file: "temp.rq",
  path:  ".../test_match/priv/rdf/queries/temp.rq",
  type: :rdf,
  uri: "file:///.../test_match/priv/rdf/queries/temp.rq"
}
